# Sp02-retention-analytics
📌 O Desafio
No cenário de um dos maiores CDs da América Latina, a alta rotatividade de novos colaboradores (as "remessas") gera custos elevados de treinamento e perda de produtividade. O objetivo deste dashboard é entender: Por que os talentos estão saindo e onde a operação está "quebrando" o novato?

📊 Principais Métricas (KPIs)
Taxa de Retenção Geral: Percentual de colaboradores que permanecem ativos após o período crítico de integração.

Índice de Talentos (Rep 2): Conversão de colaboradores operacionais para cargos de confiança.

Atrito por Setor: Identificação de áreas críticas (ex: Setor Pique) onde o esforço físico impacta diretamente na permanência.

Pareto de Motivos: Análise qualitativa que separa saídas por "Baixo Desempenho" de saídas por "Causas Estruturais" (Carga horária/Adaptação).

🛠️ Tecnologias Utilizadas
Python (Pandas): Limpeza de dados e simulação de base estatística.

SQL: Estruturação de consultas para métricas de turnover.

Looker Studio: Visualização de dados e criação de filtros interativos.

💡 Insights Obtidos
O "Mito do Desempenho": O dashboard prova que a maioria das saídas não é por falta de competência (apenas 24%), mas sim por fatores de estrutura e esforço físico.

Setores de Risco: O setor de Pique apresenta uma taxa de desligamento muito superior, sugerindo a necessidade de rotação de tarefas.
